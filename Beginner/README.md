# Beginner Level Project

## Project Title
Data Visualization using Python

## Description
This project demonstrates the fundamentals of data visualization using Python. It includes different types of charts such as Line Plot, Bar Chart, Scatter Plot, Histogram, and Pie Chart to analyze data effectively.

## Tools and Libraries
- Python
- Pandas
- Matplotlib

## Visualizations
- Line Plot
- Bar Chart
- Scatter Plot
- Histogram
- Pie Chart
