# Advanced Level Project

## Project Title
Netflix Data Analysis

## Description
This project analyzes the Netflix Movies and TV Shows dataset using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to understand trends in content type, ratings, genres, release years, and countries.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Count Plot
- Rating Distribution
- Content by Country
- Release Year Trend
- Heatmap

## Dataset
Netflix Titles Dataset
