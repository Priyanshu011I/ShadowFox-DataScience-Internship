# Intermediate Level Project

## Project Title
Air Quality Index (AQI) Data Analysis

## Description
This project analyzes Air Quality Index (AQI) data using Python. It includes data cleaning, exploratory data analysis (EDA), and data visualization to identify air pollution trends.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Line Plot
- Bar Chart
- Scatter Plot
- Histogram`

## Dataset
AQI Dataset (.csv)
